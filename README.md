# Task Forge

Task Forge is a React practice project built as a polished todo app. It includes task creation, editing, completion toggles, delete actions, search, filtering, and localStorage persistence.

## What is included

- React + Vite project structure
- Seeded sample tasks with real values for screenshots and demos
- Add, edit, complete, delete, and clear-completed actions
- Search and task filters for all, active, and completed items
- Responsive UI tuned for desktop and mobile

## Run it locally

```bash
npm install
npm run dev
```

## Verify it

```bash
npm run build
npm run lint
```

## Notes for submission

- The app is already preloaded with example tasks so the board looks complete on first run.
- Task data persists in the browser using `localStorage`.
- You can swap the seed data in `src/App.jsx` if you want different submission values.
